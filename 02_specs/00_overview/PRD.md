# PRD.md

> 状态：2026-06-20 当前实现口径
> 位置：`02_specs/00_overview/PRD.md`

## 1. 项目概述
- 产品类型：多人在线异步协作实验平台
- 研究目标：在可控流程中记录 A/B 协作、AI 辅助、任务2干预与阶段切换行为
- 当前重点：在 HTTPS 域名部署基础上，完成最终材料、指导语、问卷、变量导出与上线前稳健性收口

## 2. 用户与角色
### 2.1 参与者
- 参与者前台统一显示为 `A` / `B`
- 系统内部和导出仍保留 `A/B` 角色码；旧“尽调员 / 投资经理”只作为历史说明或导出 displayRole 兼容字段存在
- 参与者不再由 admin 预设角色，而是先按进入顺序成组，再在组内随机分配 `A/B`

### 2.2 管理员
- 上传合格手机号名单
- 配置实验时长与 AI 等级
- 配置休息问卷模板与测试题模板
- 查看 session、导出实验数据

## 3. 核心业务规则
### 3.1 自动配对
- admin 只上传合格手机号，不上传角色
- 系统先按进入顺序每 `2` 人组成一组
- 组内角色在配对完成时再随机决定
- 角色随机化使用可审计 seed，支持回溯

### 3.2 公司顺序
- 每组共享一条 session 内随机、无放回、可复现的正式公司顺序
- A 按共享正式公司顺序推进
- B 不机械跟随固定顺序，而是在共享正式公司池上动态分配：优先接收 A 已提交、B 未完成、B 未分配的 locked pool 公司；锁定池为空时 fallback 到 PreA 公司；都没有时进入空窗
- 不再使用稳健/成长、简单/复杂等维度派单

### 3.3 实验结构
- 指导语后先进入同步准备页，再进入测试题；双方都通过后才启动测试轮
- 测试轮完成后再进入第二次同步准备页，随后进入正式实验
- 正式实验为 `3` 个工作段 + `2` 个休息问卷段
- 测试轮默认单独做 `1` 家练习案例，最终版测试轮公司为 `P37`
- 最终正式材料库为 `P01-P36`，测试轮真实任务2数量固定为 0；任务2教学题仅为前端临时演示题
- 工作段默认 `20` 分钟，休息问卷段默认 `5` 分钟，可由 admin 全局调整
- 顶栏在主工作台、测试轮、休息问卷页、副线展开页统一显示当前阶段倒计时

### 3.4 A/B 协作机制
- A 每家公司有固定 `5` 分钟窗口
- A 的 `5` 分钟到时自动保存，并解锁该公司的 A 信息给 B
- B 不必等 `5` 分钟后才开始，可以先看自己的材料、写自己的判断、使用主线 AI
- 在 A 信息解锁前，B 的提交按钮保持禁用
- A 信息解锁后，B 提交按钮即可用；是否点开 A 信息区作为行为记录保留，但不再作为提交门槛
- A 信息解锁瞬间，系统保存一次 B 端“5分钟快照 + 时间戳”
- 若工作段先结束，则主线冻结，下一工作段恢复剩余公司进度

### 3.5 任务2机制
- 顶部任务2入口固定为“任务2”，不显示具体数字
- 滚动文案固定为“任务2有新题目，请及时处理”
- 滚动方向为从右往左，滚完后等待 `30s` 再次滚动
- 点击“任务2”或滚动消息都进入任务2展开页
- 任务2展开页复用主工作台布局：左材料、右上作答、右下 AI
- 任务2已完成累计不跨工作段继承；进入新工作段时清空并重新积累
- `continuous / batch` 操纵的是提醒频率，不是题目实际到达速度；后台默认都按固定节奏持续到达

### 3.6 AI 机制
- AI 等级按工作段 `1/2/3` 配置为 `basic` 或 `advanced`
- 任务1与任务2在同一工作段内共用同一 AI 等级
- 任务1 AI 上下文按公司隔离；切换公司后前端上下文清空，但数据库保留完整历史
- 任务2 AI 上下文不随公司切换而清空，按“参与者 + 当前实验 + 当前工作段”持续
- `basic` 不支持图片上传与图片粘贴；`advanced` 支持图片能力

## 4. 当前范围内的关键功能
- 手机号登录与自动配对
- 指导语页、同步准备页、测试题页与测试轮工作台
- 正式 A/B 主工作台
- 3 工作段 / 2 休息问卷段的阶段切换与恢复
- 顶部任务2入口与任务2展开页
- 任务1 AI 与任务2 AI
- 正式案例 / 测试轮案例双层材料系统接入与 A/B 主工作台表单
- admin 基础配置、测试题模板、休息问卷模板与数据导出
- 完整行为日志、快照与元数据采集

## 5. 当前不进入本轮范围
- 浏览器截图按钮
- 通用材料后台
- 高保真 Word/PDF 排版还原
- 复杂研究分析报表实时计算

## 6. 成功标准
- 参与者可从登录一路走到测试轮、正式实验、结束页
- 自动成组、组内随机角色、固定公司顺序、A `5` 分钟解锁、B 在 A 解锁后提交等核心规则真实生效
- admin 可上传名单、调整阶段参数、查看基础运行态
- 任务1、任务2、AI、问卷、快照和日志口径一致，支持后续分析
