（二）最终建议标准
标准建议：不投资

研究者理由：
虽然公司存在真实机会，包括新一代高压芯片进入客户验证后期、关键客户导入推进、试单反馈偏积极，以及新产品单品毛利空间高于老产品所带来的单位成本下降和单品毛利改善线索，但当前阶段仍存在重点风险，即新产品量产稳定性未坐实。该风险会进一步放大普通风险中体现的供应链交期/工艺波动、现金流回款压力和核心客户采购规模下调、订单兑现低于预期等问题。因此，本公司应适用重要风险一票否决规则，标准建议为“不投资”。

说明：

（一）可判为“有效备注”的情形
1. 提示跨材料关联
例：新产品验证推进是机会，但交付稳定性和客户放量需要合并判断。

2. 提示可信度差异

3. 提示 B 优先核验事项
例：若时间有限，优先核验交付稳定性和客户后续放量。

4. 提示尚未完全坐实但值得注意的线索
例：封测与量产环节仍可能存在工艺收敛问题。

（二）不判为“有效备注”的情形
1. 仅重复表格中已填写内容；
2. 只有情绪性评价，没有信息增量；
3. 与材料无关的主观臆测；
4. 过于空泛，如“整体不错”“建议关注风险”。

十、A 评分口径（研究者用）

说明：
A 的评分不是看其是否独立写出第五部分的重点机会/重点风险，也不是看其是否写出第七部分的最终投资建议标准，而是看其前台填写的尽调表与标准尽调表的匹配情况。

（一）主要评分依据
1. 基础事实摘录命中情况；
2. 隐性信号命中情况（标准答案为“是”，A 实填为“是”）；
3. 是否误报不存在的隐性信号（标准答案为“否”，A 实填为“是”）。

（二）不纳入 A 硬评分的内容
1. 第五部分中的重点机会、重点风险、普通机会、普通风险分类；
2. 第七部分中的最终投资建议标准；
3. 第九部分中的备注内容。

十一、B 报告评分与证据口径

说明：
1. 本部分用于 B 报告、团队绩效和最终投资建议的后台评分；
2. B 不需要看到本节内容；
3. B 报告不要求逐句引用材料编号，但关键主张应能被 A 尽调表或原始材料支持。

（一）最终建议正确性

1. 若 B 给出“不投资”或“暂不投资”，且理由覆盖“生产线异常/交付不稳”这一重点风险，则 RecommendCorrect 可判为 1。
2. 若 B 给出“投资”，原则上 RecommendCorrect 判为 0。
3. 若 B 给出“不投资”但理由完全未覆盖生产线异常、良率、返工、交付稳定性等重点风险，仅泛泛提及“风险较多”或“财务不好”，最终建议方向可按正确处理，但 Coverage、RiskCoverage、BEvidenceSupport 应降低。

（二）关键覆盖点

B 报告中应重点覆盖以下信息：

1. 重点风险：生产线异常/交付不稳；
2. 普通风险：关键供应链交期/工艺波动；
3. 普通风险：核心客户采购规模下调/订单兑现低于预期；
4. 普通风险：现金流紧张/回款压力；
5. 重点机会：新产品/新技术接近研发成功或验证通过；
6. 普通机会：关键客户导入推进/试单反馈积极；
7. 普通机会：单位成本下降/单品毛利改善线索。

（三）无证据支持主张

以下情形可计入 UnsupportedClaim 或 BUnsupported：

1. 报告称公司已获得重大政策、牌照或认证催化，但材料未支持；
2. 报告称公司已经签署重要客户大额合同、续约或扩单，但材料只支持试单、验证或导入推进；
3. 报告称公司存在重大诉讼、仲裁、监管处罚、负面舆情或高管离职，但材料未支持；
4. 报告称产能释放或扩产进展顺利，但材料未支持；
5. 报告称客户一定会大规模放量，但材料只支持验证推进、试单反馈积极和订单兑现条件尚未完全满足；
6. 报告直接作出超出材料证据的确定性判断。

十二、约束后续材料正文的原则

1. 第四部分中标准答案为“是”的信号，必须在原始材料中存在可被识别的支持线索；
2. 第四部分中标准答案为“否”的信号，不得在正文中出现足以支持“明确存在”的表述；
3. 第五部分中的重点机会、重点风险、普通机会、普通风险，必须能够由第四部分中的隐性信号上推得到；
4. 第六部分中的材料定位、后台功能、可支持题号和写作边界，必须落实到 P01-1 至 P01-8 的正文中；
5. 第七部分中的最终建议标准，必须能够由第五部分解释，而不能依赖正文之外的信息；
6. P01-1 至 P01-8 正式材料定稿后，应将第六部分表格中的概括性证据替换为材料中的实际原句、数据或截图文字；
7. 不得让 A 与 B 在默认情况下做成同一件事；
8. 不得让关键短板主要表现为 A 显著变慢或 B 长时间空等；
9. 本公司材料包设置 P01-1 至 P01-8，共 8 份参与者可见材料；不另设 P01-9。
